# DevFlow CLI

DevFlow CLI is a lightweight command-line utility that helps developers quickly bootstrap projects.

## Features

- Generate README files
- Create MIT License
- Generate .gitignore
- Display project tree
- Random project name generator

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python -m devflow readme
python -m devflow license
python -m devflow gitignore
python -m devflow tree
```
